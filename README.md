# Orbit Discord Bot

**Orbit** is a modular Discord bot built in Go, designed to handle server interactions like welcoming new members, music playback, and more.

---

## Features

- **Welcome Messages**: Sends a personalized welcome message when a new member joins.  
- **Music Playback**: Play audio from YouTube links or search queries directly in voice channels using `/play`.  
- **Slash Commands**: Supports intuitive Discord slash commands for easy use.  
- **Lightweight and Modular**: Built with Go and DiscordGo for performance and maintainability.
